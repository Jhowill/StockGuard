# EstoqueGuard Offline

Base inicial do app mobile offline para controle de estoque.

## Stack

- Expo SDK 54
- React Native
- TypeScript
- Expo Router

## Scripts

```bash
npm run start
npm run typecheck
```

## Estado atual

Esta versão já inclui:

- navegação base com Expo Router;
- onboarding;
- tabs principais;
- telas iniciais de produtos, alertas, relatórios, configurações, categorias, fornecedores, premium e backup;
- tema dinâmico com base em sistema, claro e escuro;
- i18n local em `pt-BR`, `en` e `es`;
- componentes visuais reutilizáveis.
- pacote de ícones já aplicado pela pasta `icons/` na config do Expo.

Banco local, anúncios, rewards e persistência ainda ficam para a próxima etapa.
